drums | distributed robot monitoring system

drums is a tool for monitoring and debugging distributed robot systems, and a complement to robot middleware systems. drums provides online time-series monitoring of the underlying resources that are partially abstracted away by middleware like ROS. Interfacing with the middleware, drums provides deabstraction and demultiplexing of middleware services to reveal the system-level interactions of your controller code, the middleware, operating system and the robot(s) environment.
